This directory contains Ox codes for the article "Bayesian analysis of verbal autopsy data using factor models with age- and sex-dependent associations between symptoms".

BF-AS.ox: code for estimation of distributions of causes of death using the proposed model.

relevance.ox: code for evaluation of the relevance of predictors to causes of death using the proposed model.

VA-BF.ox: code for estimation of distributions of causes of death using Bayesian factor models by Kunihama et al. (2020), which selects the number of factors based on the cross validation method.

See the pdf file (description.pdf) for more detail.

Ox is a fast matrix language for statistical analysis (Doornik, 2007), and it can be downloaded from the official website (http://www.doornik.com/download.html).
